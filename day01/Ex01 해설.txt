- 클래스이름은 저장이름과 무조건 같아야 한다.
- 클래스이름의 첫글자는 무조건 대문자이다.
- 클래스는 자바에서 저장할 수 있는 최소 단위이다.
- 클래스는 메소드와 필드로 구성되어 있다.
- 필드 : 상태값 = 데이터, 아는 것
- 메소드 : 동작 = 하는 것
- 필드 === 변수와 상수

public static void main(String[] args){}
- ()는 메소드이다. (ex 이름() )
- main() : 메인메소드는 JVM이 호출해서 사용하는 메소드임. 반드시 1개만 존재

System.out.println("자바의 꿈");
- System.out => 모니터를 말함(표준출력장치)
- System.in   => 키보드를 말함(표준입력장치)
